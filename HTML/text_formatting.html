  • Strong:

    If we need to stress the importance of any word in a piece of text, we use the <strong> tag. Example: When I entered the room, I felt a hot breeze swiftly pass by my ear. Note that <strong> tag is for semantic purpose and for accessibility, it makes the text bold.


  • Emphasize:

    If we need to emphasize on any word in particular but not too strong, then we use the <em> tag. Example: Gauri said that Monika had to complete the work by tommorrow. Note that <em> tag is again for semantic purpose and accessibility, it makes the text italic.


  • Bold:

    Now, <b> tag is used to make the enclosed content bold. This might look bold, just like in <strong> tag. But semantically, it is used only when visually we want to draw attention, but for accessibility: such as screen readers, this is insignificant. Example: You should complete this in another 10 minutes.


  • Italic:

    Now, <i> tag is to denote text or content that is not of the main language, technical or special words, etc.. (just as a visual cue). Example: MNREGA is a scheme introduced by the Indian Government that guarantees employment to agricultural labourers for 200 days in a year.


  • Highlight:

    If we want to highlight a part of a text, then we use the <mark> tag. Example: We strongly believe that Engineering Degree is a waste of time.


  • Strikethrough:

    If we want to have strikethrough, then we have the <del> tag. Example: This is a strikethrough.


  • Superscript:

    For superscript, we have the <sup> tag. Example: 1/10th of 100 is 10.


  • Subscript:

    For subscript, we have the <sub> tag. Example: H2 + O2 -> H2O


  • Small:

    To show a small piece of text, we use the <small> tag. Example: This is a small piece of text.
