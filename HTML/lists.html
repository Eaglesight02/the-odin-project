This html file is about the use of ordered and unordered lists

Lists:

Unordered Lists:

We create an unordered list with the <ul> tag and including the <li> tag for each list element within the unordered list.

An example can be given for the below shopping list:

  • Apples
  • Oranges
  • Guavas
  • Eggplant
  • All mix spices

Notice that we only get filled circles as the pointers, but we can change that using styling, to perhaps: squares

  • Apples
  • Oranges
  • Guavas
  • Eggplant
  • All mix spices

Fancy a hollow circle?

  • Apples
  • Oranges
  • Guavas
  • Eggplant
  • All mix spices

That is enough for unordered lists, now let's go to Ordered lists.


Ordered Lists:

We create an ordered list with the <ol> tag and include the <li> tag for each list element within the ordered list.

An example can be taken for the same grocery list:

  1. Apples
  2. Oranges
  3. Guavas
  4. Eggplant
  5. All mix spices

Now, we can change the order to maybe, decreasing (strictly)?

  1. Apples
  2. Oranges
  3. Guavas

Perhaps we want to start our ordering from a pre-determined number?

  1. Apples
  2. Oranges
  3. Guavas

Note: We can also create horizontal lists:

To display lists in horizontal fashion, we either use inline-block or inline value for the display property. Using inline-block, we can add vertical lines, so it is more preferred. When inline-block or inline is used, the list element marker - which is a circle, disc, square, number, etc... is removed. Here, we are including margin of 0 for top and bottom and 10px for left and right.

For unordered list:

  • Apples
  • Oranges
  • Guavas
  • Eggplant
  • All mix spices

For ordered list:

  1. Apples
  2. Oranges
  3. Guavas
  4. Eggplant
  5. All mix spices

As we can see above, there is no change or difference, as the list element marker is removed.

If we want the list marker, then we can use the float property instead of the inline-block or inline properties

  • Apples
  • Oranges
  • Guavas


Description Lists:

A description list is a list of having terms and their descriptions.

We create a description list using the <dl> tag with <dt> as the description term and <dd> as the description element.

Apples
Apples are good for your health
Oranges
They are high in Vitamin C
Watermelon
They are rich in water content and mostly eaten during summers
